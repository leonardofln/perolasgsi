Pérolas BETA
============

# Instruções de instalação:

- clonar o projeto para o diretório de páginas do seu servidor de aplicação (ex.: www, public_html e etc)
- renomear o diretório para "teste"
- editar o arquivo /teste/application/config/database.php
	- informar os dados de conexão com o MySQL
		- hostname
		- username
		- password
		- database
- aplicar os SQLs do arquivo /teste/dados.sql no banco de dados
- acessar o endereço: http://localhost/teste/
- para logar com o papel de administrador:
	- login: administrador@gmail.com
	- senha: teste123
- para logar com o papel de usuário:
	- login: usuario@gmail.com
	- senha: teste123
